# Claude Tutorial Project

## Purpose
This project is a learning resource for Claude Code mastery.
It demonstrates skills, hooks, agents, commands, memory, and MCP configuration.

## Stack
- Platform: macOS (darwin)
- Shell: zsh
- Package manager: pnpm (prefer over npm/yarn)

## Rules
- Always read a file before editing it
- Run tests after any code change
- Prefer explicit over implicit code
- No emojis in output unless asked
- Lead with the answer, not the explanation
